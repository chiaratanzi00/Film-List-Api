---


## Descrizione del progetto

### Gestione Completa dei Film (CRUD)

- Aggiunta di un nuovo film:
  - titolo
  - regista
  - voto
  - data
  - orario
  - commento
- Modifica di un film tramite **modal** dedicata
- Eliminazione di un film con **finestra di conferma**
- Visualizzazione del commento in un **modal**
- Lista dei film generata dinamicamente via JavaScript


### Ricerca & Filtri

- Ricerca in tempo reale per **titolo**
- Ricerca in tempo reale per **regista**


### Grafico Interattivo (Chart.js)

- Grafico di tipo **line chart** che mostra i voti dei film
- Aggiornamento automatico del grafico ad ogni modifica della lista (aggiunta, modifica o eliminazione di un film)


### API

L‚Äôapp utilizza **MockAPI** per la gestione dei dati tramite REST API:

- `GET` ‚Äì Recupero lista film
- `POST` ‚Äì Aggiunta di un nuovo film
- `PUT` ‚Äì Modifica di un film esistente
- `DELETE` ‚Äì Rimozione di un film
